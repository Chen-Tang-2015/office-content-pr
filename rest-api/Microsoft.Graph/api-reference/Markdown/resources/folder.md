# folder resource type

A folder in a user's mailbox, such as Inbox, Drafts, and Sent Items. Folders can contain messages and other folders.

### JSON representation

Here is a JSON representation of the resource

```json
{
  "childCount": 1024
}

```

|:---------------|:--------|:----------|
