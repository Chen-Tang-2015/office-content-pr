# identity resource type

The Identity type represents an identity of an _actor_. For example, and actor can be a user, device, or application.

### Properties
| Property	   | Type	|Description|
|:---------------|:--------|:----------|
|displayName|String|The identity's display name. Note that this may not always be available or up to date.
