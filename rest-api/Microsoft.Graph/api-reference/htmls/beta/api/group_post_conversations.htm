Create Conversation

Use this API to create a new Conversation.

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /groups/<objectId>/conversations
POST /users/<objectId>/joinedGroups/<objectId>/conversations
POST /drive/root/createdByUser/joinedGroups/<objectId>/conversations

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, supply a JSON representation of Conversation object.

Response

If successful, this method returns 201, Created response code and Conversation object in the response body.

Example

Request

Here is an example of the request.

POST https://graph.microsoft.com/beta/groups/<objectId>

In the request body, supply a JSON representation of Conversation object.

Response

Here is an example of the response.

Content-type: application/json
