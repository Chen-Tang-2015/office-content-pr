Create SubscribedSku

Use this API to create a new SubscribedSku.

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /siteCollections

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, supply a JSON representation of SubscribedSku object.

Response

If successful, this method returns 201, Created response code and SubscribedSku object in the response body.

Example

Request

Here is an example of the request.

POST https://graph.microsoft.com/beta/siteCollections

In the request body, supply a JSON representation of SubscribedSku object.
