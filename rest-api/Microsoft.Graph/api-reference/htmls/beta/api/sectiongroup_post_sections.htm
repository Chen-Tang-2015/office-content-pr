Create Section

Use this API to create a new Section.

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /users/<objectId>/notes/sectionGroups/<id>/sections
POST /groups/<objectId>/notes/sectionGroups/<id>/sections
POST /drive/root/createdByUser/notes/sectionGroups/<id>/sections

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, supply a JSON representation of Section object.

Response

If successful, this method returns 201, Created response code and Section object in the response body.

Example

Request

Here is an example of the request.

POST https://graph.microsoft.com/beta/users/<objectId>/notes/sectionGroups/<id>

In the request body, supply a JSON representation of Section object.

Response

Here is an example of the response.
