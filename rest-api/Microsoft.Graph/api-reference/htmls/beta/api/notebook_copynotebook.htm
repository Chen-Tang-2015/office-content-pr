notebook: copyNotebook

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /users/<objectId>/notes/notebooks/<id>/copyNotebook
POST /groups/<objectId>/notes/notebooks/<id>/copyNotebook
POST /drive/root/createdByUser/notes/notebooks/<id>/copyNotebook

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, provide a JSON object with the following parameters.

Parameter Type Description
siteCollectionId String
siteId String
renameAs String

Response

If successful, this method returns 200, OK response code and CopyNotebookModel object in the response body.
