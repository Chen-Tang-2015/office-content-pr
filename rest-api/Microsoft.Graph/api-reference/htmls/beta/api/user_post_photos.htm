Create photo

Use this API to create a new photo.

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /users/<objectId>/photos
POST /drive/root/createdByUser/photos
POST /drive/root/lastModifiedByUser/photos

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, supply a JSON representation of Photo object.

Response

If successful, this method returns 201, Created response code and Photo object in the response body.

Example

Request

Here is an example of the request.

POST https://graph.microsoft.com/beta/users/<objectId>

In the request body, supply a JSON representation of Photo object.

Response

Here is an example of the response.

HTTP/1.1 201 Created
Content-type: application/json
Content-length: 53

{
  "height": 99,
  "width": 99,
  "id": "id-value"
}
