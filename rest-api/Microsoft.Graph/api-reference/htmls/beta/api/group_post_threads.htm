Create thread

Use this API to create a new thread.

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /groups/<objectId>/threads
POST /users/<objectId>/joinedGroups/<objectId>/threads
POST /drive/root/createdByUser/joinedGroups/<objectId>/threads

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, supply a JSON representation of ConversationThread object.

Response

If successful, this method returns 201, Created response code and ConversationThread object in the response body.

Example

Request

Here is an example of the request.

POST https://graph.microsoft.com/beta/groups/<objectId>

In the request body, supply a JSON representation of ConversationThread object.
