user: findMeetingTimes

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /users/<objectId>/findMeetingTimes
POST /drive/root/createdByUser/findMeetingTimes
POST /drive/root/lastModifiedByUser/findMeetingTimes

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, provide a JSON object with the following parameters.

Parameter Type Description
attendees AttendeeBase
locationConstraint LocationConstraint
timeConstraint TimeConstraint
meetingDuration Duration
maxCandidates Int32
isOrganizerOptional Boolean

Response

If successful, this method returns 200, OK response code and MeetingTimeCandidate object in the response body.
