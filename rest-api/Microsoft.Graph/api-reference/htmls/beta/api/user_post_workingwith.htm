Create User

Use this API to create a new User.

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /users/<objectId>/workingWith
POST /drive/root/createdByUser/workingWith
POST /drive/root/lastModifiedByUser/workingWith

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, supply a JSON representation of User object.

Response

If successful, this method returns 201, Created response code and User object in the response body.

Example

Request

Here is an example of the request.

POST https://graph.microsoft.com/beta/users/<objectId>

In the request body, supply a JSON representation of User object.
