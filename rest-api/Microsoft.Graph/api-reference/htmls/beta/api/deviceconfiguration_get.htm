Get deviceConfiguration

Retrieve the properties and relationships of deviceconfiguration object.

Prerequisites

The following scopes are required to execute this API:

HTTP request

GET /deviceConfiguration/<objectId>

Optional query parameters

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

Do not supply a request body for this method.

Response

If successful, this method returns a 200 OK response code and deviceConfiguration object in the response body.

Here is an example of the request.

GET https://graph.microsoft.com/beta/deviceConfiguration/<objectId>
