Create version

Use this API to create a new version.

Prerequisites

The following scopes are required to execute this API:

HTTP request

POST /drive/root/versions
POST /drive/items/<id>/versions
POST /drives/<id>/root/versions

Request headers

Name Type Description
X-Sample-Header string Sample HTTP header. Update accordingly or remove if not needed

Request body

In the request body, supply a JSON representation of item object.

Response

If successful, this method returns 201, Created response code and item object in the response body.

Example

Request

Here is an example of the request.

POST https://graph.microsoft.com/beta/drive/root

In the request body, supply a JSON representation of item object.
