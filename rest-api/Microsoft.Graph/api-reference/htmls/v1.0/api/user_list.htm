List users

Retrieve a list of user objects.

Prerequisites

One of the following scopes is required to execute this API: User.Read.All; User.ReadWrite.All; Directory.Read.All; Directory.ReadWrite.All; Directory.AccessAsUser.All; User.Read; User.ReadWrite; User.ReadBasic.All

HTTP request

GET /users

Optional query parameters

Name Value Description
$orderby string Comma-separated list of properties that are used to sort the order of items in the response collection.
$top int The number of items to return in a result set.

Request headers

Header Value  
Authorization Bearer %token%  
Accept application/json  

Request body

Do not supply a request body for this method.

Response

If successful, this method returns a 200 OK response code and collection of User objects in the response body.

Example

Request

Here is an example of the request.

GET https://graph.microsoft.com/v1.0/users
