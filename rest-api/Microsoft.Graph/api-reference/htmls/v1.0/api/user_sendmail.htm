user: sendMail

Prerequisites

One of the following scopes is required to execute this API: Mail.ReadWrite; Mail.Send

HTTP request

POST /users/<objectId>/sendMail

Request headers

Header Value
Authorization Bearer %token%
Content-Type application/json

Request body

In the request body, provide a JSON object with the following parameters.

Parameter Type Description
message Message
saveToSentItems Boolean

Response

If successful, this method returns 200, OK response code. It does not return anything in the response body.

Example

Here is an example of how to call this API.

Request

Here is an example of the request.

POST https://graph.microsoft.com/v1.0/users/<objectId>/sendMail
Content-type: application/json

{
  "message": {
    "subject": "Meet for lunch?",
    "body": {
      "contentType": "Text",
      "content": "The new cafeteria is open."
    },
    "toRecipients": [
      {
        "emailAddress": {
          "address": "garthf@contoso.onmicrosoft.com"
        }
      }
    ]
  },
  "saveToSentItems": "false"
}
Response

Here is an example of the response.

HTTP/1.1 200 OK
