ContactFolder resource type

A folder that contains contacts.

JSON representation

Here is a JSON representation of the resource

{
  "ChildFolders": [
    {
      "@odata.type": "microsoft.graph.contactfolder"
    }
  ],
  "Contacts": [
    {
      "@odata.type": "microsoft.graph.contact"
    }
  ],
  "DisplayName": "String-value",
  "Id": "String-value (identifier)",
  "ParentFolderId": "String-value"
}

Properties

Property Type Description
DisplayName String The folder's display name.
Id String Unique identifier of the contact folder. Read-only.
ParentFolderId String The ID of the folder's parent folder.

Relationships

Relationship Type Description
ChildFolders ContactFolder collection The collection of child folders in the folder. Navigation property. Read-only. Nullable.
Contacts Contact collection The contacts in the folder. Navigation property. Read-only. Nullable.

Tasks

Task Return Type Description
Get ContactFolder ContactFolder Read properties and relationships of contactFolder object.
Create ContactFolder ContactFolder Create a new ContactFolder by posting to the ChildFolders collection.
Create Contact Contact Create a new Contact by posting to the Contacts collection.
Update ContactFolder Update ContactFolder object.
Delete None Delete ContactFolder object.
